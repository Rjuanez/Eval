Descripción detallada

Representa un problema en la plataforma con sus estadisticas de cuantos envios se han hecho, cuantos envios exitosos y el ratio de envios/envios_exitosos del problema. Ademas de un identificador unico.

void Problema::sumar_envio ( int  r)

Actualiza las estadisticas despues que un usuario haga un envio a este problema.

Precondición

Postcondición
Se actualizan las estadisticas del parametro implicito, en caso de que r = 1, se entiende que el envio es correcto y el problema queda resuelto para ese usuario
